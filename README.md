Sistema completo de gestão financeira pessoal e empresarial integrado ao WordPress.

== Description ==

O First Financial Box é um plugin WordPress completo para gestão financeira pessoal e empresarial. Todos os dados ficam no seu banco de dados WordPress, com segurança e privacidade totais.

**Recursos:**
* Dashboard com KPIs, fluxo de caixa e despesas por categoria
* Transações bancárias com filtros avançados e paginação
* Importação de extratos OFX e CSV (Mercado Pago, Nubank, Banco Inter e genérico)
* Contas a Pagar/Receber com leitor de código de barras FEBRABAN
* NFC-e modelo 65 (SEFAZ-SP): consulta, importação e análise de produtos
* Orçamento mensal por categoria com progress bars
* Relatórios DRE e Fluxo de Caixa com gráficos Chart.js
* Permissões por role do WordPress

== Installation ==

1. Faça upload da pasta `first-financial-box` em `/wp-content/plugins/`
2. Ative o plugin em WordPress → Plugins
3. Acesse o menu **Financial Box** no painel do WordPress
